Project Overview

This project involved creating a detailed spaceship with realistic sci-fi design. The goal was to master hard surface modeling and PBR texturing techniques.

Process & Techniques

The spaceship was created using several advanced techniques:

  • Polygonal modeling with subdivision surface
  • UV unwrapping for precise texturing
  • PBR textures (Albedo, Normal, Roughness, Metalness)
  • HDRI lighting for realistic rendering
